# Minecraft-JE-Panda-Server-Cilent-Modpack
Minecraft JE 大熊猫服务器的客户端，虽然原版Minecraft什么模组也不装也可以玩服务器，但是装一些实用模组会使得游戏体验更好

## 如何下载？
1. 点击绿色<font color=green>Code</font>按钮,再点击Download ZIP以下载整合包
2. 下载之后，请解压ZIP文件。

## 如何安装？
1. 打开zip
2. 如果你的电脑上已经有PCL启动器了并在运行，请先关掉它
3. 双击打开里面的`Plain Craft Launcher.exe`文件
4. 根据提示进行操作并等待安装。
